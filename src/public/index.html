⚡ Como funciona

O servidor roda 24/7 e a cada 10 minutos:

1. Busca carrinhos abandonados na Yampi

2. Calcula o tempo de abandono

3. Decide o template ideal:

0 a 1.5h → Lembrete Gentil
1.5 a 12h → Apoio + Confianca
12 a 36h → Prova Social
36 a 96h → Urgencia + Cupom

4. Nunca repete o mesmo template para o mesmo carrinho

5. Dispara via WhatsApp Cloud API
