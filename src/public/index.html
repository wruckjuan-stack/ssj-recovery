⚡ Como funciona

O servidor verifica a cada 15 minutos:

1. Busca carrinhos na Yampi

2. Filtra os que tem last_transaction_status de PIX ou boleto expirado

3. Dispara template adequado:

PIX expirado → Template "pix_expirado"
Boleto vencido → Template "boleto_expirado"

4. Nunca repete envio para o mesmo carrinho

⚡ Como funciona

O servidor roda 24/7 com 3 automacoes:

🛒 Carrinho abandonado (a cada 10 min)
0 a 1.5h → Lembrete Gentil
1.5 a 12h → Apoio + Confianca
12 a 36h → Prova Social
36 a 96h → Urgencia + Cupom
💳 PIX/Boleto expirado (a cada 15 min)
PIX expirado → Mensagem com novo link
Boleto vencido → Mensagem com novo link
🔄 Recompra (diario as 10h)
30 dias → Cupom 10% OFF
60 dias → Cupom 15% OFF
90 dias → Cupom 20% OFF
